# Beerpoly Homepage
It is a open source code of Beerpoly home page.
```
cd $GOPATH/src/github.com/gameinpl
git clone https://github.com/gameinpl/beerpoly-home.git
cd beerpoly-home
goatcli build
go run ./main run
```
